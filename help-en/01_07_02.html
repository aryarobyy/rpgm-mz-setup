Controls via the Context Menu

Right-clicking Map Data items in the Map List allows you to do things such as change settings and copy. The functions of each item are as follows.

  • Edit

    Opens the Map Properties window. For more information regarding items that can be configured, please see [Map Properties].

  • New

    Adds new map data. For more information regarding items that can be configured, please see [Map Properties].

  • Load Sample Map

    New map data is created based on sample data. Click the data name on the list in the displayed window, confirm the details, and click [OK].

  • Copy

    Copies map data to the clipboard.

  • Paste

    Adds map data currently in the clipboard.

  • Delete

    Deletes map data.

  • Shift

    Shifts the placement of all tiles on the map. Specify the shift direction and number of tiles.

  • Generate Dungeon

    Automatically generates a maze form map. Please see [How to Use Aid Tools] for more information.

  • Save as Image

    Saves the currently selected map in an image format.

Group Maps

By dragging map data in the Map List onto other map data, the data will appear at the end of the latter's map data. Just like grouping the interior maps of buildings under the map of a town, it is convenient for grouping and managing map data. By dragging a map which has been moved to the end of the list to the folder containing the project name, you can move it to the top.

This hierarchical display only changes the way that things are displayed on the Map List. This will in no way affect the design or settings of the map.
