Terms Settings

This Data's Role

Terms are the collection of names of things such as game commands and parameters. By changing the standard names to fit your game's world, you can create an even more original game.

You can enter characters in the space provided in each setting's field. However, the entire name may not be displayed if it is too long.

Parameter Details

  • Basic Statuses

    Names for Level, HP, MP, TP, and EXP. Specify the words used in the settings provided for each default name. Parameters with [(abbr.)] are where you can specify a shortened version to be displayed in places such as the status window in the Battle screen.

  • Parameters

    Names for parameters. Specify the words used in the settings provided for each default name.

  • Equipment Type

    Names for the locations of equipment. Specify the words used in the settings provided for each default name.

  • Commands

    The names of commands and choices displayed in places like the game's Menu Screen. Specify the words used in the settings provided for each default name.

  • Messages

    The contents of messages displayed in the game. Specify the text used in the settings provided for each default. Parts such as "%1" and "%2" will be replaced as appropriate when run.
