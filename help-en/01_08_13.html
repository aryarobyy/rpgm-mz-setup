Type Settings

This Data's Role

Type data is the collection of name settings for game elements, skill types, weapon types, armor types, and equipment types. The types configured here will be reflected in the Type section for each settings screen.

Parameter Details

  • Elements

    The list of element names. Click the name you want to change in the list, and enter the name in the field at the bottom. To increase or decrease the total number of names, click the [Change Maximum] button, then specify the new amount.

    This name will be used whenever making a selection in the editor. Changing the specific details of elements is done using skill and weapon data.

  • Skill Type

    The pre-classification list specified in [Skill Types] in Skill Data. The method for changing names is the same as the [Elements] section.

  • Weapon Types

    The pre-classification list specified in [Weapon Types] in Weapon Data. The method for changing names is the same as the [Elements] section.

  • Armor Types

    The pre-classification list specified in [Armor Types] in Armor Data. The method for changing names is the same as the [Elements] section.

  • Equipment Type

    The pre-classification list specified in [Equipment Types] in Armor Data. The method for changing names is the same as the [Elements] section.
