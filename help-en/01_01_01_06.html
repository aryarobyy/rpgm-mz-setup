Game Control Improvements

RPG Maker MZ also improves on functions from a player perspective. Players will more easily understand the appeal of the games you create.

Improved Smartphone UI

UI improvements were implemented to allow for easier play even when using touch controls with a smartphone or other device.

  • Optimized menus for dual PC/smartphone use
  • Scroll function via window swipe
  • Touch-selectable targets in battle

Official plugins usable with touch controls have also been included. You can create games centered more around touch controls.
Please see [Documentation] → [Using Official Plugins] for more information.

Autosave Function

The topmost save slot will be used as the autosave slot and the game will be automatically saved at the end of a battle and when the player is transferred. Even if the game is suspended for some reason, the player can restart from the autosave.

The autosave function can be disabled from the settings in [Database] → [System 1]'s [Options].

Distinction Between Creations via "Game ID"

In RPG Maker MZ, it is possible to distinguish between creations using Game ID. Save data can be transferred when a game published in the browser is updated even when the game title is changed.
You can check the game ID from [Database] → [System 2] → [Advanced Settings].
